# 프로젝트 목록

## 프로젝트 주의사항

- 불필요한 코드 삭제

- 테스트 코드 필수

- 일관된 코드 컨벤션 유지하기

  - 별도의 정해진 컨벤션은 없지만 모든 프로젝트에서 코드 컨벤션이 동일해야함

    ```javascript
    // example
    if ( true ) {
        console.log(true);
    }
    
    for ( var i = 0; i < 5; i++ ) {
        console.log(i);
    }
    ```

    - Tab은 4칸, 조건절/반복절 띄워쓰기등등
